# korus-studio

Offline music rehearsal studio built with HTML, CSS and JavaScript.

Kórus Studio nasceu com um objetivo simples: criar uma ferramenta offline, rápida e acessível para músicos, cantores e instrumentistas ensaiarem e se apresentarem sem depender de serviços na nuvem. Cada versão representa um passo na construção de uma plataforma feita por músicos, para músicos.
